# Cosmic Events

A dark, high-contrast Obsidian theme inspired by the vastness of space.

## Features

- **True Black Background**: Deep space dark for minimal distraction.
- **Vibrant Syntax Highlighting**: Glowing colors for headings, inline code, and symbols.
- **Nebula Accents**: Soft splashes of cyan, violet, and sky blue.

## Installation

You can install this theme directly within Obsidian if it's available in the community theme directory.

Alternatively, to manually install:
1. Download `obsidian-true-black.css` from this repository.
2. Place it in your vault's `.obsidian/themes/` folder (or `.obsidian/snippets` if using it as a snippet).
3. Select "Cosmic Events" (or ensure the snippet is enabled) in your Obsidian settings.
